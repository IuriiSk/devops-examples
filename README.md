# DevOps Examples

## Как использовать
- `k8s/` — манифесты для GKE
- `helm/` — чарты для Django/Node.js
- `prometheus/` — мониторинг и логи
- `scripts/` — бэкапы в DO Spaces, создание Droplet, GKE cluster

## Технологии
✅ Kubernetes (GKE), Docker, LXC  
✅ Helm, Ansible  
✅ Prometheus, Loki, Grafana  
✅ Jenkins Pipeline  
✅ MySQL, MongoDB, Redis  
✅ Python, Node.js, Next.js  
✅ Nginx Ingress
